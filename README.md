# banometruWeb

Website pentru Gala Nova Signal — proiect realizat în cadrul competiției Banometru by ING.

## Stack

- Next.js 16 (App Router, static export)
- Tailwind CSS 4
- Framer Motion
- TypeScript

## Cum modifici conținutul

Editează `content.json` din rădăcina proiectului — toate textele, prețurile, datele și membrii echipei sunt definite acolo.

## Cum adaugi imagini

Pune fișierele în `/public/images/` (fotografii echipă în `/public/images/team/`) și actualizează referințele din `content.json`.

## Dezvoltare locală

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
